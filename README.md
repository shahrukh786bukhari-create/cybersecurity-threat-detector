# 🔐 Cybersecurity Threat Detector

A Python-based tool to detect brute-force login attacks by monitoring failed login attempts.

## 🚀 Features
- Detects brute-force attacks
- Tracks IP-based login attempts
- Alerts on suspicious activity

## 🧠 How It Works
The system monitors login attempts and flags any IP address with repeated failed logins.

## ▶️ Run the Project
python detector.py
